Package topologies

source code

This package contains ready-made topologies for quick experiments. The topologies can be customised via parameters or inheritance. Topologies contain the code that defines which Mininet hosts exist in an experiment and how they are interconnected.

Submodules [hide private]
  • topologies.AbstractTopology: This module defines the abstract base interface for topologies.
  • topologies.BriteTopology: This file defines a topology that reads a random network generated with BRITE and creates a Mininet topology from it.
  • topologies.BriteTopologyTest
  • topologies.GameoverTopologyTest: This file tests a topology where 6 Bots talk to 2 CnC servers via a number of Proxies.
  • topologies.LayeredTopology: This file defines a layered Topology.
