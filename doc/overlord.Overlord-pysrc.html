  3  """This file defines a class that centrally controls every Host in the network. This includes all sorts of actors. 
  4  It is also responsible for triggering random events such as bot desinfections, traffic generation, etc. 
  5  It uses unix sockets to communicate with the clients because they are independent of the network communication in Mininet. 
  6   
  7  The Overlord communicates with the hosts via remote procedure call (RPC) over Unix (file) sockets. 
  8  Unix sockets are files that allow to separate processes to communicate similar to a network connection but without using the 
  9  actual network infrastructure of the operating system. It was necessary to have the Experiment and Hosts separated in this way, 
 10  because mininet only allows to run separate processes in its nodes. It is not possible to run threads on mininet hosts. 
 11  RPC allows a smooth integration with regular Python code. Unix sockets were chosen because they do not interfere with the Mininet 
 12  network and they will still be working, even if the particular host is unable to use the network, 
 13  which may happen during some experiments."""
